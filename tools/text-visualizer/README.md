## Usage

1. **Enter Your Text**: Type or paste your text in the input area
2. **Choose a Pattern**: Select from 6 available distribution patterns
3. **Adjust Typography**: Fine-tune font, size, weight, and spacing
4. **Configure Pattern**: Use pattern-specific controls to perfect your layout
5. **Apply Transforms**: Add rotation and opacity effects
6. **Export**: Download your creation as a PNG image

## Technical Details

- **Zero Dependencies**: Pure HTML/CSS/JavaScript implementation
- **Canvas-Based**: Uses HTML5 Canvas 2D API for rendering
- **Real-Time Preview**: Live updates as you adjust parameters
- **High-DPI Support**: Crisp rendering on retina displays

## Tips & Tricks

1. **For Logos**: Use the circle pattern with "Apply to each character" for dynamic logo designs

3. **For Art**: Combine spiral pattern with varying opacity for depth effects

5. **For Backgrounds**: Random pattern with low opacity for texture overlays
